PHP web based Password Manager for business and personal use.

### New Features & Modernization (Fork)
- [x] **PHP 8.1+ Compatibility**
- [x] **OIDC Authentication (Azure AD, Google Workspace, Keycloak)**
- [x] **REST API alongside JSON-RPC**
- [x] **Docker, Docker Compose and CI/CD (GitHub Actions) ready**
- [x] **Dark Theme (material-dark)**
- [x] **Modern Security (Argon2id, RSA 2048 OAEP, bcrypt, Secure Cookies)**
- [x] **Password expiry notifications**
